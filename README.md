# Portfolio - John DOE

The goal of this project is to create and launch the website for an aspiring web developer, John DOE.
The site is built using HTML and CSS, with the Bootstrap framework integrated via React.js.
A minimal amount of custom CSS is used, primarily for visual effects.

The final site mockups were provided as working materials.

---

## Tech Stack

- **React.js** (Create React App)
- **Bootstrap 5**
- **React Router DOM**

---

## Installation

### Prerequisites

- Node.js
- Git

### Steps

1. **Clone the repository**
```bash
git clone <https://github.com/Marine-Briet/portefolio.git>
cd portefolio
```

2. **Install dependencies**
```bash
npm install
```

---

## Running the app
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

---

## Production build
```bash
npm run build
```

Optimized files are generated in the `build/` folder.
